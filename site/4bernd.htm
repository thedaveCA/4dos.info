Bernd Schneider's Batches

 

On this page I have collected the (mostly) game batches of Bernd Schneider. Bernd suddenly appeared out of nowhere (for me :-) ) and sent me a bunch of amazing batches, mostly games I wouldn´t have dreamed to write in 4DOS.

 

Further games have their origin in my needling him to try his hand on games I thought would be impossible to do in 4DOS, e. g. Hanoi, Solitaire, Sokoban.

 

I´ve heard him mumbling about a better chess batch, so I won´t be overly surprised if he produces a chess batch to challenge IBM´s Deep Thought next. If he could find it in him to write a few comments and institute some kind of version control to make my life easier ...

 

Bernd writes his btms exclusivly in 4DOS (and in caps and with a 01-01-93 date. Don´t ask me why!). That means 4NT users have to adapt a few things (mostly DOS pipes to temp files, sometimes character probs due to Unicode).

 

Bernd´s internet access is rather limited, b.t.w., so I do this collection for him pro temp.

 

4DOS

 

  • ABRECH         Batch to keep score at Skat, a german card game

  • BOMB           Something like Minesweeper

  • CHASE          A board game

  • CLEAN          Bernd's version of my XCLEAN to delete unwanted files

  • DEFRAG         Intelligent HD defragmenter :-)

  • DICE           The game known as Yatzee (german Kniffel)

  • FALL           A strategic challenge

  • FFD            Utility: find file by date

  • FORTWO         A Skat (german card-) game

  • HANGMAN        Bernd's modified version of my Hangman

  • HANOI          "The Towers of Hanoi" Batch to play the game or to let the computer solve the well-known problem. 3 to seven disks. A real gem.  

  • HIHO            Solitaire, the board game, with a  good user interface, replay, and a perfect solution to he puzzle

  • KILL            Run and evade Pacman style game

  • KNIFFEL         A game involving dice :-)

  • LABY            A Labyrinthian game

  • LITES           A strategy game

  • LIZZY           An Eliza style Psychotherapist

  • MEMO            Memory, the game.

  • MIND            Bernd's correction to my Mastermind clone

  • ORACLE          An oracle that really could change your life!

  • POKER           Just that. Don´t loose your last Penny!

  • PUSH            Use strategic thinking to solve these puzzlers

  • PUZZLE          A sliding puzzle. Again a very nice batch.

  • PYRAMID         A Solitaire-type card game

  • SHIP            Sink the subs and battleships. The old schoolgame played on checkered paper.

  • SOKO            Sokoban: push those crates to their right places. A real feat, external layers, so it´s possible to create your own! A must-have.

  • SOLI            Solitaire, the card game. Under Windows thsi was the biggest productivity killer in many offices

  • VOKABEL         English trainer

  • WOLF            A simple board game: can the wolves catch the sheep?

  • 4MOUSE.COM     is a DOS mouse driver to use at the command line and in batches. Used in some of these batches.

 Some updates to files from my 4XBTM collection by Bernd:

  • ASCII           update of XASCII.ZIP, an ascii table
  • CALENDA         update of XCALAE.BTM, a nice calendar batch
  • CHANGES         update of XCHANGES.BTM, a batch to check for free memory changes
  • CONVERT         update of XCONVERT.BTM, a dec / hex / bin number converter
  • FFD             update of XFFD.BTM, a file find utility
  • FSIZE           update of XFSIZE.BTM, a utility to add filesizes
  • KEYTEST         update to XKEYTEST.BTM, showing ascii value and keynumber
  • MASTER          Bernd´s version of my Mastermind batch.
  • MOON            update of XMOON.BTM, showing the moon's phases for a given date.
