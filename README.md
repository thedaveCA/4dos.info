# 4dos.info

Deployment via Cloudflare Pages, published to [4dos.info](https://4dos.info), with selected files published to [files.4dos.info](https://files.4dos.info/). Requests are redirected as needed.

File extensions listed in `site-vs-files.txt` will be routed to `files.4dos.info`, and `transform-rule.txt` contains a [Cloudflare Redirect rule](https://dash.cloudflare.com/7b6dd6309ba48f3062c2be9f1918a174/4dos.info/rules/redirect-rules).
